# 🚀 Startup Resource Management System (SRMS)

A professional web-based Startup Resource Management System built using **Flask, Python, Bootstrap, JSON storage, and multithreading**.  
It helps startups manage **team members, tasks, resources, and reports** through a centralized dashboard.

## 📌 Features

### 📊 Dashboard
- Total Team Members
- Active Tasks
- Completed Tasks
- Resource Availability
- Real-time statistics

### 👥 Team Management
- Add members
- View members
- Delete members
- Department & role tracking

### 📋 Task Management
- Create tasks
- Assign tasks to team members
- Set priority levels
- Mark tasks as completed
- Track task progress

### 📦 Resource Management
- Add resources
- Track availability
- Monitor usage

### 📈 Reports Module
- Project overview
- Team summary
- Task analysis
- Resource monitoring

## ▶️ How to Run

### 1. Install Flask
```bash
pip install flask
```

### 2. Run the application
```bash
python app.py
```

### 3. Open in browser
```
http://127.0.0.1:5000
```
